Sobre Mim

Meu nome é Ricardo Chaves de Aragão Marques e estou no 3º semestre de Ciência da Computação na UFRGS. Desde que comecei a estudar programação, desenvolvi afinidade com automação de processos e linguagens como Python e C. Trabalhei como estagiário no Instituto Rio-Grandense do Arroz (junho/2021 – junho/2022), onde criei planilhas de custos e realizei análise de séries temporais para apoiar decisões estratégicas. Em seguida, assumi a função de gerente comercial em uma loja de eletrônicos (junho/2022 – maio/2024), liderando equipe, cuidando de atendimento a clientes e desenvolvendo o site da empresa. Essas experiências reforçaram minha capacidade de comunicação, empatia e resolução de problemas. Paralelamente, desenvolvi projetos pessoais em Python, C/C++, HTML/CSS e JavaScript. Entre eles, destaco: Jogo Snake em C com Raylib, com fases, alimentos especiais e ranking de recordes; Scripts de Web Scraping em Python para coleta de preços em e-commerce; API Financeira em Python para consulta de dados de mercado; Tour Virtual em 360° com hotspots usando Pannellum; Ferramenta em Node.js para gerar páginas estáticas otimizadas para SEO. Tenho conhecimentos em Git/GitHub, Microsoft Office, além de inglês avançado e espanhol intermediário. Meu objetivo é atuar como desenvolvedor júnior, aprofundando-me em desenvolvimento web full-stack ou outra área e entregando soluções de qualidade em equipe.
